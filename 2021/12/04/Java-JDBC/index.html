事务处理

Connection

可以完成数据库相关的操作。

  • setAutoCommit()：false开启事务；
  • commit()：提交事务；
  • rollback()：回滚；

Statement

执行SQL命令，不安全不能防止SQL注入。

  • executeQuery(String sql)：执行查询；
  • executeUpdate(String sql)：更新、插入、删除；
  • execute(String sql)：执行任何SQL；

PreparedStatement

执行SQL命令，效率高，可以防止SQL注入。

ResultSet

返回的结果集。

  • beforeFirst()：移动到最前面；
  • afterLast()：移动到最后面；
  • next()：移动到下一个；
  • getObject()：获取元素（不知道类型）；
  • absolute(int row)：移动到指定行；
