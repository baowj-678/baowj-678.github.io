主要实现类

HashMap

声明

特点

  • 线程不安全；
  • 可以存储null的key和value；
  • 底层是数组+链表+红黑树；

put

  • 放入key-value；

  • 先调用key.hashCode()获取哈希值；

  • 利用哈希值计算在table（Node类型）数组中的存储位置；
    • 如果该位置为空；
    • 如果该位置不为空：
      • 如果key的哈希值和某个数据的哈希值一样，继续调用equals()方法比较；
        • 如果返回true：替换value；
        • 如果返回false：添加成功；
      • 否则，如果该位置数据为tree：
        • 向tree中添加元素key-value；
      • 否则，依次判断链表每个结点的哈希值和equals()是不是等于key：
        • 如果不存在则创建新的节点：
          • 如果链表节点总数超过8个，则将链表转成红黑树；
        • 如果存在，则替换value；

LinkedHashMap

特点

  • 继承自HashMap；
  • 可以按照添加顺序进行遍历【添加了指针】；

声明

TreeMap

特点

  • 底层是红黑树；
  • 需要实现排序；

Hashtable

特点

  • 线程安全，效率不高；
  • 不能存储null；

Properties

特点

  • 继承自Hashtable；
  • 常用来处理配置文件；
  • key和value都必须是String；
